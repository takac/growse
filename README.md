# growse

A program to open the GitHub page, Bitbucket, Gitlab for a repository.


| Backend   | Repo | Branch | File Path | File Path with Line No. |
|-----------|------|--------|-----------|-------------------------|
| GitHub    | Yes  | Yes    | Yes       | Yes                     |
| Bitbucket | Yes  | Yes    | No        | No                      |
| Gitlab    | Yes  | No     | No        | No                      |
| Others    | No   | No     | No        | No                      |
